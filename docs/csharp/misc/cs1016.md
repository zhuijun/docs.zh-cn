# <a name="compiler-error-cs1016"></a><span data-ttu-id="ebcbb-103">编译器错误 CS1016</span><span class="sxs-lookup"><span data-stu-id="ebcbb-103">Compiler Error CS1016</span></span>

<span data-ttu-id="ebcbb-104">应为命名特性参数</span><span class="sxs-lookup"><span data-stu-id="ebcbb-104">Named attribute argument expected</span></span>  
  
 <span data-ttu-id="ebcbb-105">未命名特性参数必须出现在命名参数之前。</span><span class="sxs-lookup"><span data-stu-id="ebcbb-105">Unnamed attribute arguments must appear before the named arguments.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ebcbb-106">示例</span><span class="sxs-lookup"><span data-stu-id="ebcbb-106">Example</span></span>  

 <span data-ttu-id="ebcbb-107">以下示例生成 CS1016：</span><span class="sxs-lookup"><span data-stu-id="ebcbb-107">The following sample generates CS1016:</span></span>  
  
```csharp  
// CS1016.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Class)]  
public class HelpAttribute : Attribute  
{  
    public HelpAttribute(string url)   // url is a positional parameter  
    {  
        m_url = url;  
    }  
  
    public string Topic = null;      // Topic is a named parameter  
    private string m_url = null;  
}  
  
[HelpAttribute(Topic="Samples", "http://intranet/inhouse")]   // CS1016  
// try the following line instead  
//[HelpAttribute("http://intranet/inhouse", Topic="Samples")]  
public class MainClass  
{  
    public static void Main ()  
    {  
    }  
}  
```
