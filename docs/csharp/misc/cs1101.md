# <a name="compiler-error-cs1101"></a><span data-ttu-id="8019e-103">编译器错误 CS1101</span><span class="sxs-lookup"><span data-stu-id="8019e-103">Compiler Error CS1101</span></span>
<span data-ttu-id="8019e-104">参数修饰符“ref”不能与“this”一起使用。</span><span class="sxs-lookup"><span data-stu-id="8019e-104">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="8019e-105">当 `this` 关键字修改静态方法的第一个参数时，它向编译器发出信号，指示该方法是扩展方法。</span><span class="sxs-lookup"><span data-stu-id="8019e-105">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="8019e-106">不需要或不允许对扩展方法的第一个参数使用任何其他修饰符。</span><span class="sxs-lookup"><span data-stu-id="8019e-106">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8019e-107">示例</span><span class="sxs-lookup"><span data-stu-id="8019e-107">Example</span></span>  
 <span data-ttu-id="8019e-108">下面的示例生成 CS1101：</span><span class="sxs-lookup"><span data-stu-id="8019e-108">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
