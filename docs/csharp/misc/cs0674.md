<span data-ttu-id="9e207-104">不要使用“System.ParamArrayAttribute”。</span><span class="sxs-lookup"><span data-stu-id="9e207-104">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="9e207-105">请改用“params”关键字。</span><span class="sxs-lookup"><span data-stu-id="9e207-105">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="9e207-106">C# 编译器不允许使用 <xref:System.ParamArrayAttribute?displayProperty=nameWithType>；请改用 [杂注](../language-reference/keywords/params.md) 。</span><span class="sxs-lookup"><span data-stu-id="9e207-106">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="9e207-107">以下示例生成 CS0674：</span><span class="sxs-lookup"><span data-stu-id="9e207-107">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
