## <a name="example"></a><span data-ttu-id="0f885-106">示例</span><span class="sxs-lookup"><span data-stu-id="0f885-106">Example</span></span>

```csharp
// CS0162.cs
// compile with: /W:2
public class Program
{
    public static void Main()
    {
        goto lab1;
        {
            // The following statements cannot be reached:
            int i = 9;   // CS0162
            i++;
        }
    lab1:
        {
        }
    }
}
```
