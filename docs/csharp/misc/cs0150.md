# <a name="compiler-error-cs0150"></a><span data-ttu-id="03518-103">编译器错误 CS0150</span><span class="sxs-lookup"><span data-stu-id="03518-103">Compiler Error CS0150</span></span>

<span data-ttu-id="03518-104">应输入常量值</span><span class="sxs-lookup"><span data-stu-id="03518-104">A constant value is expected</span></span>  
  
 <span data-ttu-id="03518-105">在应发现常量的位置处发现变量。</span><span class="sxs-lookup"><span data-stu-id="03518-105">A variable was found where a constant was expected.</span></span> <span data-ttu-id="03518-106">有关更多信息，请参见 [switch](../language-reference/keywords/switch.md)。</span><span class="sxs-lookup"><span data-stu-id="03518-106">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="03518-107">以下示例生成 CS0150：</span><span class="sxs-lookup"><span data-stu-id="03518-107">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="03518-108">当数组大小由变量值指定并由数组初始值设定项初始化时，也会产生此错误。</span><span class="sxs-lookup"><span data-stu-id="03518-108">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="03518-109">若要删除该错误，在单独语句或多个语句中初始化此数组。</span><span class="sxs-lookup"><span data-stu-id="03518-109">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
