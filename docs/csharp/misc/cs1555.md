# <a name="compiler-error-cs1555"></a>编译器错误 CS1555
无法找到为“Main”方法指定的“class”  
  
 为 [-main](../language-reference/compiler-options/main-compiler-option.md) 编译器选项指定了类，但在源代码中找不到类名。
