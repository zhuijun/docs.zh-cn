# <a name="compiler-error-cs0568"></a><span data-ttu-id="ab6b1-103">编译器错误 CS0568</span><span class="sxs-lookup"><span data-stu-id="ab6b1-103">Compiler Error CS0568</span></span>
<span data-ttu-id="ab6b1-104">结构不能包含显式的无参数构造函数</span><span class="sxs-lookup"><span data-stu-id="ab6b1-104">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="ab6b1-105">每个 [结构](../language-reference/builtin-types/struct.md) 都有一个无参数的构造函数，该构造函数将对象初始化为零。</span><span class="sxs-lookup"><span data-stu-id="ab6b1-105">Each [struct](../language-reference/builtin-types/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="ab6b1-106">因此，你可以为结构创建构造函数，该构造函数必须采用一个或多个参数。</span><span class="sxs-lookup"><span data-stu-id="ab6b1-106">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="ab6b1-107">下面的示例生成 CS0568：</span><span class="sxs-lookup"><span data-stu-id="ab6b1-107">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
