# <a name="compiler-error-cs0617"></a><span data-ttu-id="048f4-103">编译器错误 CS0617</span><span class="sxs-lookup"><span data-stu-id="048f4-103">Compiler Error CS0617</span></span>

<span data-ttu-id="048f4-104">“reference”不是有效命名的特性实参，因为它不是有效的特性形参类型</span><span class="sxs-lookup"><span data-stu-id="048f4-104">'reference' is not a valid named attribute argument because it is not a valid attribute parameter type</span></span>  
  
 <span data-ttu-id="048f4-105">尝试访问特性类的 [私有](../language-reference/keywords/private.md) 成员。</span><span class="sxs-lookup"><span data-stu-id="048f4-105">An attempt was made to access a [private](../language-reference/keywords/private.md) member of an attribute class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="048f4-106">示例</span><span class="sxs-lookup"><span data-stu-id="048f4-106">Example</span></span>  

 <span data-ttu-id="048f4-107">下面的示例生成 CS0617。</span><span class="sxs-lookup"><span data-stu-id="048f4-107">The following sample generates CS0617.</span></span>  
  
```csharp  
// CS0617.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Struct |
                AttributeTargets.Class |  
                AttributeTargets.Interface)]  
public class MyClass : Attribute  
{  
   public int Name;  
  
   public MyClass (int sName)  
   {  
      Name = sName;  
      Bad = -1;  
      Bad2 = -1;  
   }  
  
   public readonly int Bad;  
   public int Bad2;  
}  
  
[MyClass(5, Bad=0)] class Class1 {}   // CS0617  
[MyClass(5, Bad2=0)] class Class2 {}  
```
