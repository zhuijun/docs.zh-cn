# <a name="compiler-warning-level-1-cs0184"></a>编译器警告（等级 1）CS0184
给定表达式始终不是所提供的（“type”）类型  
  
 该表达式绝不会是 **true** ，因为你测试的变量既不声明为“类型” ****** 也不派生自“类型” ******。  
  
 下面的示例生成 CS0184：  
  
```csharp  
// CS0184.cs  
// compile with: /W:1  
class MyClass  
{  
   public static void Main()  
   {  
      int i = 0;  
      if (i is string)   // CS0184  
         i++;  
   }  
}  
```
