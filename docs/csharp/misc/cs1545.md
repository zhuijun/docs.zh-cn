# <a name="compiler-error-cs1545"></a><span data-ttu-id="1d9a4-103">编译器错误 CS1545</span><span class="sxs-lookup"><span data-stu-id="1d9a4-103">Compiler Error CS1545</span></span>
<span data-ttu-id="1d9a4-104">属性、索引器或事件“property”不受现用语言支持；请尝试直接调用访问器方法“set accessor”或“get accessor”</span><span class="sxs-lookup"><span data-stu-id="1d9a4-104">Property, indexer, or event 'property' is not supported by the language; try directly calling accessor methods 'set accessor' or 'get accessor'</span></span>  
  
 <span data-ttu-id="1d9a4-105">该代码正在使用具有非默认 [索引器](../programming-guide/indexers/index.md) 的对象并试图使用索引的语法。</span><span class="sxs-lookup"><span data-stu-id="1d9a4-105">The code is consuming an object that has a non-default [indexer](../programming-guide/indexers/index.md) and tried to use the indexed syntax.</span></span> <span data-ttu-id="1d9a4-106">若要解决此错误，请调用该属性的 `get` 或 `set` 访问器方法。</span><span class="sxs-lookup"><span data-stu-id="1d9a4-106">To resolve this error, call the property's `get` or `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1d9a4-107">示例</span><span class="sxs-lookup"><span data-stu-id="1d9a4-107">Example</span></span>  
  
```cpp  
// CPP1545.cpp  
// compile with: /clr /LD  
// a Visual C++ program  
using namespace System;  
public ref struct Employee {  
   Employee( String^ s, int d ) {}  
  
   property String^ name {  
      String^ get() {  
         return nullptr;  
      }  
   }  
};  
  
public ref struct Manager {  
   property Employee^ Report [String^] {  
      Employee^ get(String^ s) {  
         return nullptr;  
      }  
  
      void set(String^ s, Employee^ e) {}  
   }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="1d9a4-108">示例</span><span class="sxs-lookup"><span data-stu-id="1d9a4-108">Example</span></span>  
 <span data-ttu-id="1d9a4-109">下面的示例生成 CS1545：</span><span class="sxs-lookup"><span data-stu-id="1d9a4-109">The following sample generates CS1545.</span></span>  
  
```csharp  
// CS1545.cs  
// compile with: /r:CPP1545.dll  
  
class x {  
   public static void Main() {  
      Manager Ed = new Manager();  
      Employee Bob = new Employee("Bob Smith", 12);  
      Ed.Report[ Bob.name ] = Bob;   // CS1545  
      Ed.set_Report( Bob.name, Bob);   // OK  
   }  
}  
```
