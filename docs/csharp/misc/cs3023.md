## <a name="example"></a><span data-ttu-id="c7f54-106">示例</span><span class="sxs-lookup"><span data-stu-id="c7f54-106">Example</span></span>  

 <span data-ttu-id="c7f54-107">下面的示例生成警告 CS3023：</span><span class="sxs-lookup"><span data-stu-id="c7f54-107">The following example generates warning CS3023:</span></span>  
  
```csharp

public class Test  
{  
    [return:System.CLSCompliant(true)]  // CS3023  
    // Try this instead:  
    // [method:System.CLSCompliant(true)]  
    public static int Main()  
    {  
        return 0;  
    }  
}  
```
