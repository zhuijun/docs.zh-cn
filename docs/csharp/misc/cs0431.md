 <span data-ttu-id="38af9-106">同时使用了“::”与引用类型的别名。</span><span class="sxs-lookup"><span data-stu-id="38af9-106">You used "::" with an alias that references a type.</span></span> <span data-ttu-id="38af9-107">若要解决此错误，请使用“.”运算符。</span><span class="sxs-lookup"><span data-stu-id="38af9-107">To resolve this error, use the "." operator.</span></span>  
  
 <span data-ttu-id="38af9-108">下面的示例生成 CS0431:</span><span class="sxs-lookup"><span data-stu-id="38af9-108">The following sample generates CS0431:</span></span>  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer
{  
   public class Inner
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
