# <a name="compiler-error-cs0506"></a><span data-ttu-id="b6278-103">编译器错误 CS0506</span><span class="sxs-lookup"><span data-stu-id="b6278-103">Compiler Error CS0506</span></span>
<span data-ttu-id="b6278-104">“function1”：继承成员“function2”未标记为“virtual”、“abstract”或“override”，无法进行重写</span><span class="sxs-lookup"><span data-stu-id="b6278-104">'function1' : cannot override inherited member 'function2' because it is not marked "virtual", "abstract", or "override"</span></span>  
  
 <span data-ttu-id="b6278-105">已重写的方法未显式标记为 **virtual**、 **abstract**或 `override`。</span><span class="sxs-lookup"><span data-stu-id="b6278-105">A method was overridden that was not explicitly marked as **virtual**, **abstract**, or `override`.</span></span>  
  
 <span data-ttu-id="b6278-106">下面的示例生成 CS0506：</span><span class="sxs-lookup"><span data-stu-id="b6278-106">The following sample generates CS0506:</span></span>  
  
```csharp  
// CS0506.cs  
namespace MyNameSpace  
{  
   abstract public class ClassX  
   {  
      public int i = 0;  
  
      public int f()  
      {  
         return 0;  
      }  
      // Try the following definition for f() instead:  
      // abstract public int f();  
   }  
  
   public class ClassY : ClassX  
   {  
      public override int f()   // CS0506  
      {  
         return 0;  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
