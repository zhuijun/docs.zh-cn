## <a name="example"></a><span data-ttu-id="78040-111">示例</span><span class="sxs-lookup"><span data-stu-id="78040-111">Example</span></span>

## <a name="example"></a><span data-ttu-id="78040-115">示例</span><span class="sxs-lookup"><span data-stu-id="78040-115">Example</span></span>  
 <span data-ttu-id="78040-116">下面的示例生成 CS0433。</span><span class="sxs-lookup"><span data-stu-id="78040-116">The following example generates CS0433.</span></span>  
  
```csharp

using TypeBindConflicts;  
public class Test
{  
   public static void Main()
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="78040-117">示例</span><span class="sxs-lookup"><span data-stu-id="78040-117">Example</span></span>

```csharp  
// compile with: /reference:cs0433_1.dll /reference:TypeBindConflicts=cs0433_2.dll

   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();  
   }  
}  
```
