# <a name="compiler-warning-level-2-cs0467"></a><span data-ttu-id="4aafd-103">编译器警告（等级 2）CS0467</span><span class="sxs-lookup"><span data-stu-id="4aafd-103">Compiler Warning (level 2) CS0467</span></span>
<span data-ttu-id="4aafd-104">方法“method”和非方法“non-method”之间存在多义性。</span><span class="sxs-lookup"><span data-stu-id="4aafd-104">Ambiguity between method 'method' and non-method 'non-method'.</span></span> <span data-ttu-id="4aafd-105">使用方法组。</span><span class="sxs-lookup"><span data-stu-id="4aafd-105">Using method group.</span></span>

## <a name="example"></a><span data-ttu-id="4aafd-107">示例</span><span class="sxs-lookup"><span data-stu-id="4aafd-107">Example</span></span>  
 <span data-ttu-id="4aafd-108">下面的示例生成 CS0467。</span><span class="sxs-lookup"><span data-stu-id="4aafd-108">The following example generates CS0467.</span></span>  
  
```csharp  
// CS0467.cs  
interface IList
{  
    int Count { get; set; }  
}  
  
interface ICounter  
{  
    void Count(int i);  
}  
  
interface IListCounter : IList, ICounter {}  
  
class Driver
{  
    void Test(IListCounter x)  
    {  
        // The following line causes the warning. The assignment also  
        // causes an error because you can't assign a value to a method.  
        x.Count = 1;  
        x.Count(3);
        // To resolve the warning, you can change the name of the method or
        // the property.  
  
        // You can also disambiguate by specifying IList or ICounter.  
        ((IList)x).Count = 1;  
        ((ICounter)x).Count(3);  
    }  
  
    static void Main()
    {  
    }  
}  
```
