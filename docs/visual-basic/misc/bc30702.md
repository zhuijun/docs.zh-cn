# <a name="index-indexnumber-for-dimension-dimensionnumber-is-out-of-range"></a>维度“\<dimensionnumber>”的索引“\<indexnumber>”超出范围
某个索引超出定义的范围。  
  
 **错误 ID：** BC30702  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 更改过程的逻辑，以避免超出数组界限。  
  
- 重新定义数组以增加值的范围。
