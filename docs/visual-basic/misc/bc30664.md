# <a name="conversions-from-typename1-to-typename2-must-be-explicit"></a>从“\<typename1>”到“\<typename2>”的转换必须为显式转换
无法隐式实现此转换。  
  
 **错误 ID：** BC30664  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 使用 `CType`或某个特定转换函数，如 `CInt` 或 `CDec`。
