 <span data-ttu-id="0c52a-105">直接约束 `Structure` 和间接约束 `Class` 导致类型形参 `t1`冲突，原因是 `Structure` 约束要求相应的类型变量为值类型，而 `Class` 要求类型变量为引用类型。</span><span class="sxs-lookup"><span data-stu-id="0c52a-105">The direct constraint `Structure` and the indirect constraint `Class` cause a conflict for type parameter `t1`, because the `Structure` constraint requires that the corresponding type argument be a value type, while `Class` requires that it be a reference type.</span></span>
