# <a name="end-module-must-be-preceded-by-a-matching-module"></a>“End Module”前面必须是匹配的“Module”
Module 块以 `Module` 语句开头并以 `End Module` 构造结尾。  
  
 **错误 ID：** BC30622  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 请确保模块以 `Module` 语句开头。
