# <a name="method-does-not-have-a-signature-compatible-with-the-delegate"></a>方法没有与委托兼容的签名

方法的签名与你尝试使用的委托的签名不兼容。 `Delegate` 语句定义参数类型和委托类的返回类型。 任何具有匹配的兼容类型参数和返回类型的过程都可以用于创建此委托类型的实例。  
  
 **错误 ID**： BC36563
