# <a name="imetadataemit2savedelta-method"></a>IMetaDataEmit2::SaveDelta 方法
将当前的 "编辑并继续" 会话中的更改保存到指定的文件。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SaveDelta (  
    [in] LPCWSTR     szFile,
    [in] DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>参数  
 `szFile`  
 中用于保存更改的文件的名称。  
  
 `dwSaveFlags`  
 [in] 保留。 必须为零。
