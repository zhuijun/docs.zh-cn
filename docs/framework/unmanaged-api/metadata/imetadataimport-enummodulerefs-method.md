# <a name="imetadataimportenummodulerefs-method"></a>IMetaDataImport::EnumModuleRefs 方法
枚举表示导入的模块的 ModuleRef 标记。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT EnumModuleRefs (  
   [in, out] HCORENUM     *phEnum,  
   [out]     mdModuleRef  rModuleRefs[],  
   [in]      ULONG        cMax,  
   [out]     ULONG        *pcModuleRefs  
);  
```  
  
## <a name="parameters"></a>参数  
 `phEnum`  
 [in，out]指向枚举器的指针。 第一次调用此方法时，此值必须为 NULL。  
  
 `rModuleRefs`  
 弄用于存储 ModuleRef 标记的数组。  
  
 `cMax`  
 [in] `rModuleRefs` 数组的最大大小。  
  
 `pcModuleRefs`  
 弄中返回的 ModuleRef 令牌数 `rModuleRefs` 。  
  
## <a name="return-value"></a>返回值  
  
|HRESULT|说明|  
|-------------|-----------------|  
|`S_OK`|`EnumModuleRefs`已成功返回。|  
|`S_FALSE`|没有要枚举的令牌。 在这种情况下， `pcModuleRefs` 为零。|
