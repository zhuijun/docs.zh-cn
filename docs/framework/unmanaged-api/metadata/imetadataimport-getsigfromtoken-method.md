# <a name="imetadataimportgetsigfromtoken-method"></a>IMetaDataImport::GetSigFromToken 方法
获取与指定标记关联的二进制元数据签名。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetSigFromToken (
   [in]   mdSignature        mdSig,
   [out]  PCCOR_SIGNATURE    *ppvSig,
   [out]  ULONG              *pcbSig
);  
```  
  
## <a name="parameters"></a>参数  
 `mdSig`  
 中要为其返回二进制元数据签名的标记。  
  
 `ppvSig`  
 弄指向返回的元数据签名的指针。  
  
 `pcbSig`  
 弄二进制元数据签名的大小（以字节为单位）。
