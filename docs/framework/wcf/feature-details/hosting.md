# <a name="hosting"></a>Hosting
本节中的主题介绍服务承载。 可以通过 Internet Information Services (IIS) 、Windows 进程激活服务 (为) 、Windows Server AppFabric、Windows 服务或托管的应用程序来承载服务，此选项通常称为 " *自承载*"。  
  
 值得注意的是，从不受信任的主机运行服务或任何扩展会危害安全。
